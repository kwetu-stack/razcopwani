# RAZCO PWANI™
# Navbar Structure

---

## Purpose

Define the official navigation structure for RAZCO PWANI Version 1.0.

This document serves as the navigation authority for all future development.

---

# Navigation Philosophy

The menu follows the natural FMCG management workflow:

Strategy
↓
Market
↓
Execution
↓
Growth
↓
Management

This allows users to think and operate in the same sequence as the business.

---

# Mobile Navigation

Use Hamburger Menu.

Requirements:

- Fully responsive
- Bootstrap collapse navigation
- Same menu order as desktop
- One-click access to all modules

---

# Active Menu Rules

Current page must be highlighted.

Example:

If user is on:

/customers

Then:

Customers menu item = Active

---

# Future Navigation Additions

Reserved for Version 2:

- Territories
- Sales Targets
- Orders
- Inventory
- Team Management
- Notifications
- Settings

These modules must not appear in Version 1.

---

# Quick Access Rule

Dashboard must always be the first menu item.

Reports must always be the last menu item.
